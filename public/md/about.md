我选择了 arenot.cn 作为我的网站域名，这很大原因是因为它简单好记。你完全可以和我注册时想的那样记住这个域名：are not cn，这对于及时关注我的作品动向有很大帮助！（请不要过度解读它的字面意思，仅仅是为了方便好记且续费便宜❤️）

## 技术栈

此网站采用以下技术实现：

- [NuxtJS](https://nuxt.com/)
- [Vue.js](https://vuejs.org/)
- [markdown-it](https://markdown-it.github.io/)
- [highlight.js](https://highlightjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)

网站后台&数据库（此网站没有真正的后台和数据库）：

- 网站运行在 [Cloudflare Workers](https://developers.cloudflare.com/workers/) 上
- 网站所有二进制文件托管在 Github 仓库 [Nanoare-official-website](https://github.com/McdheYxY/Nanoare-official-website/tree/files/files) 的 files 分支上
- 网站所有 markdown 及评论储存在 Github 仓库 [Nanoare-official-website](https://github.com/McdheYxY/Nanoare-official-website/discussions) 的 Discussions 中
